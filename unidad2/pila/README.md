### Pila

La pila es una estructura de datos lineal en la que se puede agregar o quitar elementos por solo uno de los dos extremos. Esto provoca que los elementos dentro de una pila se eliminen el orden inverso al que se insertaron, es decir, el ultimo elemento que se agrega a la pila, es el primero en salir.
